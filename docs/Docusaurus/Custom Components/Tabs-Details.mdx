# Tabs & Details

## Tabs

Before adding tags you need to add these code line on top of your code

````js
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
````

Then you can add add this code block.

````js
<Tabs groupId = "Tab 1">
  <TabItem value="Tab 1 value" label="Tab 1 Label">

Tab 1 Content

  </TabItem>
  <TabItem value="Tab 2 value" label="Tab 2 Label">

Tab 2 Content

  </TabItem>
</Tabs>

````
